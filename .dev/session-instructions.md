# Codex Session Instructions

When working in this repository:

1. At the start of a session
   - Read `.dev/codex-context.md`
   - Read `.dev/codex-handoff.md`

2. When making meaningful progress
   - Update `.dev/codex-handoff.md`

3. When the session ends or pauses
   - Ensure `.dev/codex-handoff.md` reflects:
     - current goal
     - files modified
     - decisions made
     - next step

4. When architectural decisions are made
   - Update `.dev/codex-context.md`
