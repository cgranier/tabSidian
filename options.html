Markdown Customization

Customize the Markdown format for tab titles and URLs. Don't forget to include the {title} and {url} placeholders, and use \n for line breaks.

Default: ## {title}\n[{url}]({url})\n\n

Formatting Examples

Example 1: ## {title}\n[{url}]({url})\n\n
This will produce:
## Page Title
[https://url.com](https://url.com)

Example 2: - [{title}]({url})\n
This will produce:
- [Page Title](https://url.com)
