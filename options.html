Markdown Customization

Customize the Markdown format for tab titles and URLs. Don't forget to include the {title} and {url} placeholders in the respective fields.

Default: ## {title}
Default: [{url}]({url})

Formatting Examples

Example 1: ## {title} - [{url}]({url})
This will produce:
## Example Title - [https://example.com](https://example.com)

Example 2: * {title} ({url})
This will produce:
* Example Title (https://example.com)

Example 3:

To insert a new line, use \n in the format field:

* {title}\n({url})
This will produce:
* Example Title
(https://example.com)
