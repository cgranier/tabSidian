tabSidian - Settings

Restricted URLs

Add any URLs you want to restrict from processing by tabSidian.
Use one line per URL.

Markdown Customization

Customize the Markdown format for tab titles and URLs.
Use {title} to output the tab's Title.
Use {url} to output the tab's URL.
Use \n or press Enter for line breaks. Add a linebreak at the end, to separate your entries.


Default: ## {title}\n[{url}]({url})\n\n

Formatting Examples

Example 1: ## {title}\n[{url}]({url})\n\n
This will produce:
## Tab Title 1
[https://url.com](https://url.com)

Example 2: - [{title}]({url})\n
This will produce:
- [Tab Title 1](https://url.com) - [Tab Title 2](https://url.com)


Obsidian Export

Configure direct exports for vaults with the Advanced URI plugin installed.
Leave both fields empty to keep using local downloads only.


{timestamp} is replaced with the export timestamp. Paths must end with .md and are resolved inside your vault.
